> Welcome to termish!

A javascript based terminal for the web, by the web, to control the web...

Termish runs javscript scripts that help you access content, automate stuff,

access APIs, and whatever else you need in a safe sandboxed environment.

---

> Setting your scripts repo

You can load termish scripts from anywhere on the web.

For the moment, we have some sample scripts on this site.

You can set the repo home by doing:

termish set-repo repourl

ex: termish set-repo http://localhost:5000/repo/

Now you can install any script from this location

---

> Installing scripts from your repo

A folder in a repo will consist of a definition.json file

and one or more scripts. To install this script on your

termish, simply do:

termish install foldername

ex: termish install ghcontribs

---

> Executing scripts

And now you can run the script by giving the correct args:

ex: ghcontribs ncthis/hackertalk

---

> Listing scripts

To list all the scripts installed in your termish:

termish list

---

> Removing scripts

To remove scripts, simply do:

termish remove scriptname

ex: termish remove ghcontribs
